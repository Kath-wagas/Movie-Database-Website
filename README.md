This project is a simple movie database website that uses XML to store movie information, XPath to search and retrieve specific movie details, and XSL (XSLT) to transform and display the data as a structured web page.

The website features:

-Organized movie data stored in XML format

-Efficient searching and data retrieval using XPath

-Data transformation and presentation using XSL

-Clean and readable web page output

This project demonstrates key concepts such as:

-XML data structuring

-XPath queries for extracting specific information

-XSL transformations for rendering XML data into HTML

-Separation of data and presentation

This project serves as an introduction to XML-based web technologies and structured data presentation on the web.
